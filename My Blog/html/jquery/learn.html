jquery学习心得

jquery对象就是jquery包装Dom对象后产生的对象
jquery对象是jquery独有的
只有 jQuery 对象才能使用 jQuery 的方法，在 jQuery 对象中无法使用 DOM 对象的任何方法，反之 DOM 对象也无法使用任何 jQuery 的方法。

ready
ready()函数时jquery中非常重要的一个小函数，意思是文档加载完Dom结构后，再执行函数，执行速度快，在文档中可以出现多次；
与onload的区别是，onload在页面中只能出现一次，需要在页面所有资源加载完才执行，所以执行速度比ready慢。

jquery动画
基本隐藏、显示效果：show（）/ hide（）
若是对同一个 jQuery 对象使用，可以采用链式操作
滑动式动画效果：slideDown（）/ slideUp（）
淡入淡出：fadeIn（）淡入   fadeOut（）淡出

全局函数
全局函数实际上就是jquery本身的方法，例如$.ajax();
向 jQuery 命名空间添加一个函数，只需要将这个新函数指定为 jQuery 本身的一个属性;
$.globalFunction = function(){
// todo...
};
可以通过 jQuery.globalFunction（）或者 $.globalFunction（）来调用
当需要添加多个函数可以使用$.extend（）函数
$.extend({
functionOne: function(){
// todo...
},
functionTwo: function(){
// todo...
}
});
