Array的操作方法总结一

push 方法
该方法用于向数组obj中添加新元素，并返回新的数组
obj.push(item1,item2,.....);
obj是必选项，一个数组对象；
item1,item2.........是可选项，数组中新添加的元素；

join 方法
obj.join(separator)
obj 必选项。Array 对象。
separator 必选项。是一个 String 对象，作为最终的 String 对象中对数组元素之间的分隔符。如果省略了这个参数，那么数组元素之间就用一个逗号来分隔。
返回字符串值，其中包含了连接到一起的数组的所有元素，元素由指定的分隔符分隔开来。

unshift 方法
将指定元素插入数组开始位置，并返回新数组
obj.unshift(item1,item2,.....);
obj是必选项，一个数组对象；
item1,item2.........是可选项，数组中新添加的元素；

pop 方法
obj.pop(); 移除数组中的最后一个元素，并返回该元素
如果该数组为空，那么将返回 undefined。

shift 方法
obj.shift(); 移除数组中的第一个元素并返回该元素

concat 方法 (Array)
array1.concat(item1,item2,.........)
array1必选项，其他所有数组要进行连接的数组对象
item1,item2....可选项，要连接到array1末尾的其他对象
返回一个新数组，这个新数组是又两个或更多数组组合而成的,如果某一项为数组，那么添加其内容到 array1 的末尾。如果该项目不是数组，就将其作为单个的数组元素添加到数组的末尾。
